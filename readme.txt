== Description ==

This simple widget includes your LastFm recent tracks into the sidebar.

Enter your username and number of songs you want to show and it's ok.

== Installation ==

1. Upload the folder 'recent-last-fm-tracks' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place the widget to your desired sidebar
4. Enter your LastFm username
5. Enjoy!
